강제타입 변환
자식타입은 부모타입으로 자동변환되지만 
부모 타입은 자식타입으로 자동변환 되지 않아 강제 타입변환을 시켜야 함

Parent parent = new Child();
Child child = (Child) parent;

	    (강제타입변환)	
자식타입변수 = (자식타입) 부모타입객
	     (캐스팅 연산자)

다형성(polymorphism)이란 하나의 객체가 여러 가지 타입을 가질 수 있는 것을 의미
자바에서는 이러한 다형성을 부모 클래스 타입의 참조 변수로 자식 클래스 타입의
인스턴스를 참조할 수 있도록 하여 구현하고 있습니다.
다형성은 상속, 추상화와 더불어 객체 지향 프로그래밍을 구성하는 중요한 
특징 중 하나입니다.
