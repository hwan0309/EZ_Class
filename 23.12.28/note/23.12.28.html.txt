모든 html요소에는 두 가지의 attribute를 부여해줄 수 있다.

<a id="my>dfd</a>
#my
<a href="#" target=>"">속성</a>

*target속성
_self==> 현재 페이지에서 이동(기본값)
_parent==>부모 프레임이 있다면 부모 페이지에서 이동
_blank==>새창이 열리면서 이동(*자주 사용됨*)


-id 유일
-class 다양성, 겹칠 수 있으며 여러가지 가질 수 있음.
<a class ="b"
<div class="b"

a요소는 기본적으로 두가지 역할을 한다.
-링크로써의 기능(속성으로 href를 쓰며 값으로 해당주소 또는 #이름을 사용)
-타깃이 되는 기능(name속성을 가지는 것이나 id속성으로 대체됨)
-id속성은 모든 요소가 가질 수 있는 속성이기에 타깃을 위해 a요소 사용필요x
a요소 안에 텍스트 접근성을 위해 앞 뒤 문맥 없이도 해당 링크 페이지를 알기 쉽게
넣어야 한다.

ul> li>(필수), 순서가 없는 목록
ol> li>(필수), 순서가 있는 목록
li>블럭 요소이므로 대부분의 블럭 요소를 가질 수 있다.
dl> dt>(개념), dd>(정의), 개념 정의를 나타내는 목록

<dl>
  <dt>이름</dt>
<dl>
