
2과목 빅데이터 탐색

*단순대치법(Single Imputation)*

완전분석법
불완전 자료 모두 무시, 동계적 추론의 타당성 문제발생

평균대치법
데이터의 중앙, 평균, 최빈값 등으로 결측값 대치
결측값 발생이 다른 변수와 관계가 있는경우에 유용

단순확률 대치법
평균대치법에 적절한 확률값을 부여
Hot-deck, Cold-deck 혼합 방법이 있음
Hot-deck
현재 진행중인 연구에서 비슷한 성향을 가진 응답자의 자료로 대체
Cold-deck
외부출처 또는 이전의 비슷한 연구에서 대체할 자료를 가져옴

Outlier

*이상값 발생원인*
1.의도하지 않은 잘못입력, 측정, 실험된 데이터
2.고의적인 이상값
3.데이터 표본 설정오류
4.처리오류

*ESD (공식)*
Extreme Studentized Deviation 
평균 -3표준편차 < 정상데이터 < 평균 + 3표준편차 

사분위수 사용 (공식)
IQR = 제3사분위수 - 제1사분위수
제1사분위수 - 1.5 * IQR < 정상데이터 < Q3 + 1.5 * IQR

기하평균(공식)
기하평균 -2.5표준편차 < 정상데이터 < 기하평균 + 2.5표준편차

데이터시각화
히스토그램, 밀도차트, 상자그림, 산포도, 잔차도 등 사용

*분석기법 활용*
비지도 학습의 군집기법 사용
마할라노비스 거리(데이터 밀도(분산)을 고려한 거리)
LOF(관측치 주변의 밀도와 근접한 관측치 주변 밀도의 상대적 비교를 통해 검출)
iForest(의사결정나무를이용 분할 횟수로 이상값 검출)

*변수의 개념(Feature)*
RDBMS에서 속성, 열이라고 부르는 것 

샘플(Sample)
RDBMS에서 레코드, 행이라고 부르는 것

*변수선택기법*
필터기법(Filter)
통계적 측정방법 사용

래퍼기법(Wrapper)
변수의 일부로 모델링 후 그 결과를 확인하는 작업을 통해 성능좋은 조합을 찾는방법

임베디드 기법(Embedded)
모델자체에 포함된 변수 선택 기법

*차원축소*
차원 = 변수의 수, 차원이 증가할 수록 모델의 정확도가 떨어질 수 있음
변수의 정보를 최대한 유지하면서 변수의 개수를 줄이는 통계기법

차원의 저주
차원이 증가하면서 개별 차원내의 학습 데이터수가 차원의 수보다 적어지며
성능이 저하되는 현상

*주성분 분석*
여러 차원의 변수를 대표하는 새로운 차원의 주성분을 생성
주성분은 변수들의 선형결합으로 이루어져있음
변수들은 공분산 행렬 또는 상관행렬 사용 행의 열의 수가 같은 정방행렬에서만 사용

*성분들이 설명하는 분산의 비율*
누적 분산 비율을 확인하면 주성분들이 설명하는 전체 분산 양을 알 수 있음
누적 분산 비율이 70~90%사이가 되는 주성분 개수 선택

*Encoding 종류 및 특징*
Label Encoding 일렬번호를 사용하는 것으로 1개 숫자로 대치함

One Hot Encoding 범주의 개수만큼 변수 생성 

Binary Encoding One Hot Encoding에 비해 적은 변수 사용,
모델 학습 속도가 더 빠름

Frquency Encoding 번주형 변수를 해당 범주의 빈도로 변환

Target Encoding 종속 변수 값들의 평균을 활용

*Min-Max Normalization 공식*
(x - x최솟값)
-------------------
(x최댓값 - x최솟값

*Z-Score, standardization 공식*
(x - 전체평균)
---------------
(표준편차)

*변수를 분석하기 쉽게 변환하고자 할 때 적용하는 변환 종류*
오른쪽 꼬리가 길 때 log, 제곱근, 역수 등 (큰 값을 작게 만듬)
왼쪽 꼬리가 길 때 제곱, 지수변환 등(작은 값을 크게 만듬)

귀무가설이란 
통계적 가설 검증알 하면 귀무가설, 대립가설 설정해야함
귀무가설 = 차이가 없다, 같다 
대립가설 = 차이가 있다. 다르다
ex) 전국에 20세 이상의 평균키가 170cm라는 주장을 검정하면
귀무가설 = 20세이상의 성인 남자의 평균 키는 170cm과 같다 또는 차이가 없다
대립가설 = 20세 이상의 성인 남자의 평균 키는 170cm과 다르다 또는 차이가 있다

Cost sensitive learning은 불균형 데이터셋(imbalanced dataset)에서 유용하며
예측성능을 높이고 모델의 일반화 성능을 개선하는데 도움이 됨

*탐색적 데이터 분석의 4가지 특징*
저항성 - 수집된 자료에 결측값, 이상값이 있을때도 영향을 적게받는 성질
잔차 해석 - 잔차를 구하여 데이터의 보통과 다른 특징 탐색
자료 재표현 - 데이터 분석, 해석을 단순화 할 수 있도록 원래 변수를 변환하는 것
현시성 - 데이터 분석결과가 보기 쉽게 변환

*개별 변수 탐색*
범주형(명목, 순위 척도 데이터)
-통계값(빈도수, 최빈값, 비율, 백분율)
-시각화(bar plot(막대 그래프))
수치형(등간, 비율 척도 데이터)
-통계값(평균, 분산, 표준편차, 첨도, 왜도 등)
-시각화(histogram, boxplot)

*다차원 데이터 탐색*
범주형 - 범주형
(빈도수와 비율 활용, 막대그래프 활용)
범주형 - 수치형
(범주를 그룹으로 사용, 그룹에 따른 수치형 변수의 통계랑 차이 비교, boxplot 활용)
수치형 - 수치형
(변수간의 상관성 분석, 공분산으로 방향 파악, 상관계수로 방향, 강도파악)

*피어슨 상관계수*
대상자료는 등간척도, 비율척도 사용, 두 수 간의 선형적인 크기만 측정가능
공식
x,y의 공분산을 x,y의 표준편차의 곱으로 나눈 값
응답자1의 표준편차 2, 응답자 2의 표준편차2, 두 응답자의 공분산 값 4이면
(p) = 4 / (2*2) = 1

*스피어만 상관계수*
대상자료는 서열척도 사용, 두수간의 비선형적인 관계를 나타낼 수 있음
스피어만 상관계수는 두 데이터의 실제 값 대신 두 값의 순위를 기반으로 함
변수안의 순위가 완전 일치하면 1, 반대이면 -1

*공분산*
2개 확률변수의 선형관계를 나타내는 값
cov(x,y) < 0이면 x > E(x)일때 y < E(y)이다
두 확률변수 x,y가 독립이면 공분산 cov(x,y)은 0이지만 그 반대는 성립하지 않음

*시각화 과정*
1.정보화 구조화 - 2.정보 시각화 - 3.정보 시각표현 순서
1.유사 데이터를 묶거나 재배열, 정리 및 변환을 통해 데이터의 패턴을 찾음
수집및 탐색 - 분류 - 배열 - 재배치
2.시각화 도구를 사용해 그래프를 만들어 효과적으로 정보를 표현하는 단계
3.시각화 완성 요소단계로 그래픽 요소를 활용해 완성함

*인포 그래픽*
정보를 빠르고 정확하게 표현하기 위해 시각화로 표현
복잡한 대규모 빅데이터 분석 결과를 명료하고 이해하기 쉽게 표현함

*시각화 분류*
시간 시각화(시간 흐름에 따른 변화를 표현)
-선 그래프, 막대 그래프, 계단식 그래프, 영역차트

공간 시각화(지도를 활용하여 데이터를 표현)
-코로프레스 맵, 카토그램, 버블 플롯 맵, 등치선도, 등치지역도

관계 시각화(다변량 데이터에 대해 변수간의 연관성 및 패턴을 사용하여 분석)
-산점도, 산점도행렬, 버블차트, 히트맵

비교 시각화(다변량 데이터에 대하여 유사 또는 차이에 대해 점, 선 등을 사용하여 표현)
-평행차트, 히트맵, 스타차트, 프로팅 바 차트, 막대 그래프, 체르노프 페이스

구성 시각화(범주형 데이터의 구성을 크기로 표현)
-파이차트, 도넛차트, 트리 맵 차트

분포 시각화(연속형 데이터의 분포를 시각적으로 표현)
-1개 변수: 히스토그램, 박스플롯 포현
-2개 변수: 산점도 포현

*인포그래픽 유형*
지도형 - 특정 국가나 지역의 지도를 사용해 정보를 제공

도표형 - 다양한 표 그래프를 사용해 정보를 제공

스토리텔링형 - 줄거리라고 보일만한 요소들이 갖추어져 있고, 특정 인물이 등장해
사건에 대한 이야기를 펼치기도 함

타임라인형 - 주제를 선정하여 그와 관련된 히스토리 또는 전개 양상을 시간
흐름 형태로 나타내는 방식

비교분석형 - 두가지 이상의 제품 개념을 비교하여 정보를 제공

만화형 - 캐릭터 등의 만화적 요소를 활용

*기술 통계, 추론통계*
기술통계
포본의 속성, 특징을 파악하는 것을 목적으로 통계량 및 도표를 활용하여 의미를 도출

추론통계
모집단에서 추출된 표본으로부터 모수와 관련된 통계량의 값을 계산하고, 모집단의
특징을 알아내는 과정
분석 결과가 통계적으로 유의한지 검증하는 역할

*중심경향 통계론*
평균(Mean) 값 들의 무게 중심이 어디인지를 나타내는 값
중앙값(Median) 자료를 크기 순서대로 배열했을 때 중앙에 위치하는 값
최빈값(Mode) 어떤 값이 가장 많이 관찰되는지 나타내는 값
Mean > Median > Mode
평균은 양 꼬리 값의 크기가 변할 때 영향을 크게 받지만 중앙값은 그러한 변화에
영향을 거의 받지않음

산술평균 - 모든 자료들을 합한 후 전체 자료수로 나누어 계산
조화평균 - N개의 양수에 대하여 그 역수들을 산술평균한 것의 역수
	 -평균속도 구할때 사용
기하평균 - N개의 자료에 대해서 관측치를 곱한 후 N제곱근으로 계산
	- 물가 상승률, 경제 성장률, 인구 증가율 등의 변화율 구할때 사용

*변동계수*
표준편차를 산술평균을 기준으로 표준화 시키는 것 cv = x분의 s 
데이터가 모두 양수이면서 단위가 다른 그룹 또는 단위는 같지만 평균차이가 클 때
산포 비교에 사용

ex) A학생이 평균 3시간을 공부하였고 표준 평균은 0.4었고
    B학생이 평균 6시간 공부하고 표준편차가 0.9 이었다면 
    어떤 학생이 꾸준하게 하였을까
A = 3분의 0.4 = 0.133, 
B = 6분의 0.9 = 0.15 이므로 
변동계수가 작은 A가 더 꾸준히 공부함

*범위*
최소값과 최대값의 차이
데이터가 [1, 3, 5, 7, 10] 인 경우 범위 = 9
데이터가 [1, 3, 5, 7, 30] 인 경우 범위 = 29
이상값에 민감한 특성이 있음

*사분위 범위*
Q1(25%), Q3(75%) 일때 IQR = Q3 - Q1
중간에 있는 50%의 자료들이 갖는 값의 범위
소수 극단 값/ 이상값에 민감하지 않음

*분포 통계량 왜도/비대칭도*
-분포의 비대칭도를 나타내는 통계량으로 왜도 값이 0일때 평균과 중앙값 같음
-양의 왜도: 평균이 중앙값보다 오른쪽 위치, 오른쪽으로 꼬리가 긴 모양
-음의 왜도: 평균이 중앙값보다 왼쪽 위치, 왼쪽으로 꼬리가 긴 모양
-왜도 절대값 3 미만이면 정규분포에 가까운 정도의 데이터로 판단

*확률적 표본추출법의 종류*
단순 무작위 추출
- 모집단의 각 개체가 표본으로 선택될 확률이 동일하게 추출되는 경우

계통추출
모집단 개체에 1~n이라는 일련번호를 부여한 후, 첫번째 표본을 임의로
선택하고 일정 간격으로 다음 표본을 선택함

승화추출
모집단을 서로 겹치지 않는 층들로 나누고 각 층에서 단순 확률 표본을 추출하는 방법

군집추출
집단 내에서는 서로 이질적이며 집단 간에는 서로 동질적인 표본 추출 기법
ex( 경상대학 내에 경영학과 경제학과)
