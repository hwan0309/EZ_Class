변수란 하나의 값을 저장할 수 있는 메모리 번지에 붙여진 이름
자바 변수는 다양한 타입 (정수형, 실수형 등) 값을 저장할 수 없다.
타입에 맞는 값만 저장 가능 ex( int num = 1 o  int num = '123' x)

literal = 소스코드값에서 직접 입력된 값
정수리터럴
-기본적인 정수리터럴은 10진수로 간주한다
-0으로 시작하는 리터럴 (01, 02, 03, -04...)은 8진수로 간주한다
-정수앞에 0x로 시작하는 리터럴은 16진수로 간주한다 (0x0 ~ 0xf)

실수리터럴
-소수점이 있는 리터럴은 10진수 실수로 간주
0.12, 10.0 
-대문자 E 또는 소문자 e가있는 리터럴은 10진수 지수와 가수로 간주
5E7 -> 5X10의 7승
-실수리터럴엔 float, double가 있다

문자리터럴 
-작은 따옴표로 묶은 하나의 문자
'a','한'
-역슬레쉬(\)가 붙은 문자리터럴은 이스케이프 문자라고 한다. (특수용도 문자)
'\t' 수평탭 '\n' 줄바꿈 '\r' 리턴 '\\' = \ 
-문자열 지정은 char타입에만 할 수 있다.

문자열 리터럴
-문자열 리터럴은 쌍다옴표로 묶은 문자

논리 리터럴
-true, false
